Portal

The Portal is a Drupal 7 website that serves as a central database for ESD's school data. In addition to allowing school leaders to edit information, it also provides a read-only API for other projects (like the Scorecard) to access public data.

The site is deployed via Drupal install profile. See the code and documentation in esdportal_profile's README.
